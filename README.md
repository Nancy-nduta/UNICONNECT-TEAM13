UniConnect is a cross-platform mobile application developed with Flutter. It aims to bridge the gap between students and faculty by providing a centralized hub for academic resources, unit management, and real-time announcements.

Project Vision
To digitize the university experience, making learning materials accessible offline and ensuring students never miss a deadline through proactive notifications.

Key Objectives
Unit Organization: Allow students to view units registered per semester.

Resource Management: Provide a seamless interface for accessing PDFs, slides, and lecture links.

Communication: Enable lecturers to broadcast announcements and collect module feedback.

Task Tracking: Help students manage assignments and quiz deadlines in a single view.

Navigation Flow
The app follows a hierarchical structure typical of an Academic LMS:

Auth Screen: Secure login using Student/Staff ID.

Home Dashboard: Overview of current semester units and latest news.

Unit Detail View: Access to weekly content, assignments, and grades.

Announcements: A dedicated feed for university-wide or unit-specific alerts.

Profile/Settings: User academic details and app preferences.

Tech Stack
Framework: Flutter (v3.x).

Language: Dart.

State Management: (Plan to use Provider or Bloc for managing course data).

Version Control: Git/GitHub.
